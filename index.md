# プライバシーポリシー

## 1. 利用者情報の取得
本アプリは、ユーザーの個人情報、端末情報、位置情報、その他の利用者情報を自動的に取得・送信することはありません。

## 2. データの保存
本アプリで作成されたテキストデータや画像データは、すべてお客様の端末内のストレージ（ローカル環境）にのみ保存されます。提供者がこれらのデータを閲覧・利用することはありません。

## 3. 第三者への提供
本アプリは、法令に基づく場合を除き、利用者の同意なく第三者に個人情報や利用者データを提供することはありません。

## 4. 広告および解析ツール
本アプリには、広告配信ツールやアクセス解析ツール（Google Analytics等）は組み込まれていません。
